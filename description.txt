ТЕХНИЧЕСКОЕ ЗАДАНИЕ
"Dress Choose"

Система предназначена для индивидуального подбора одежды в соответствии с исходными данными пользователя.
Цель - обеспечить продажи элементов одежды через сервис. Изделия могут быть доступны как по предзакупкам так и через индивидуальный пошив.

I. Особенности сервиса

1. Клиент может указать параметры своей внешности
1.1 мерки на основании которых делается модель комплекции фигуры, с возможностью корректировки параметров
1.2 параметры своей внешности для определения цветотипа (цвет волос, кожи) и возрастной группы (возраст)
1.3 предпочтительные цвета
1.4 время года, для которого осуществляется подбор
1.5 дополнительные условия (удобство, комфорт при смене температуры, предпочтительный стиль, желание скрыть недостатки)
1.6 модные тенденции как ориентир для выбора комплектов (стили одежды)

2. Клиент может указать множество вещей в своем гардиробе а также то, что он хотел бы приобрести для последующего анализа.
2.1 Виртуальный гардероб
2.2 Несколько вариантов единицы одежды и что нужно выбрать для оптимального подбора сочетаний (виртуальная примерочная)

3. Сервис обеспечивает возможности
3.1 Подбор сочетаний вещей по таблице сочетаемых цветов
3.2 Определение цвета вещи по фото образца ткани
3.3 Подбор возможных вариантов образов на основании цветотипа, возраста, стиля
3.4 Подбор комплектов в привязке к базовой вещи
3.5 Просмотр вариантов различных комплектов, сохранение образа, редактирование
3.6 Подбор одежды под обувь, прическу, аксессуар
3.7 Возможность перекрашивания вещи в другой цвет для просмотра результата

Возможности сервиса позволяют
- сократить время на походы по магазинам и примерки
- улучшить внешний вид клиента благодаря выбору правильных вещей и сочетаний
- свести к минимуму покупки неудачных вещей или тех, к которым ничего не подходит, найти применение вещам которые вы не носите
- сэкономить боджет клиента поскольку не нужно будет покупать одежду в дорогих бутиках по завышенной цене, а необходимую вещь можно будет заказать у мастера
- обеспечить работой бригаду мастеров по пошиву индивидуальных заказов
- привлечь квалифицированных дизайнеров и стилистов одежды для коррективок и подбора комплектов одежды
- цена и качество изделия также будет зависеть от выбора ткани, который будет оптимизирован и соответствовать пожеланиям клиента
- при использовании сервиса не обязательно заказывать что-либо, можно получить оптимальные комплекты одежды из имеющихся вещей или из того что вы запланировали купить сами
- не нужно ждать когда в магазинах появится что-то "интересное", вы можете получить то что хотите прямо сейчас в удобное для вас время и по доступной цене
- задать интересующие вопросы квалифицированному дизайнеру или модельеру одежды
- подготовить заранее комплекты одежды и не ждать когда будут продавать что-то в сезон, сохранить образы чтобы не потерять свои идеи в будущем

II. Особенности реализации

1. Стек технологий
2. Графическая часть
3. Организация заказа через сайт, платежные системы

III. Руководство пользователя

1. Клиент регистрируется на сайте и ваходит в свою учетную запись
2. Клиент вводит свои данные (пп. I.1, I.2)
3. Клиент выбирает из возможностей сервиса (п. I.3)
4. Клиент работает с сервисом и анализирует его результаты, может сохранять результаты для последующего применения
5. Клиент может оформить заказ на покупку выбранной вещи или индивидуальный пошив и оплатить
6. Клиент получает уведомления о ходе выпонения заказа
